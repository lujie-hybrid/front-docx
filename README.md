# 前端技术文档项目

## 拉取代码

```bash
git clone http://gitlab.efreight.cn/technical-specifications-doc/front-end-doc.git
```
## 安装依赖

```bash
npm i  或者  yarn 
```
## 运行开发环境

```bash
npm run dev  或者  yarn dev
```
## 打包生产环境

```bash
npm run build  或者  yarn build

打包后的文件： /docs/.vuepress/dist
```
